=== REGRAS DE CLASSIFICAÇÃO ===

Use registrar_saida quando: gastei, paguei, comprei, saiu, gastando, fui no
Use registrar_entrada quando: recebi, entrou, caiu, ganhei, foi pago, me pagaram
Use registrar_conta_pagar quando: tenho que pagar, vence, preciso pagar, me lembra de pagar, lembrar pagar
Use registrar_conta_receber quando: tenho que receber, vão me pagar, me devem, a receber, vai me pagar
Use consultar_gastos quando: quanto gastei, meus gastos, quanto saiu, gastos de
Use consultar_entradas quando: quanto entrou, quanto recebi, minhas entradas, entradas de
Use consultar_saldo quando: qual meu saldo, como estou, quanto sobrou, meu saldo, saldo do mês
Use consultar_contas_pagar quando: contas a pagar, o que vence, boletos, pendentes para pagar
Use consultar_contas_receber quando: o que tenho a receber, quem me deve, recebimentos, quanto vou receber
Use gerar_resumo quando: resumo, como foi a semana, como foi o mês, balanço
Use corrigir_ultimo_lancamento quando: corrige, apaga o último, foi na verdade, era para ser, corrijo
Use mensagem_desconhecida quando: não for possível identificar o intent com segurança

=== REGRAS DE DATA ===

Se o usuário não informar data, use a data atual.
"hoje" = data atual
"ontem" = data atual - 1 dia
"amanhã" = data atual + 1 dia
"dia 10" = próximo dia 10 do calendário a partir de hoje
"semana passada" = semana anterior
"esse mês" / "este mês" = mês e ano atuais
Formato de saída sempre: YYYY-MM-DD
